Project Documentation

Project Overview: This project is a Minimum Viable Product (MVP) of a Network TestLab Automation tool designed to simplify and automate network device testing workflows. It simulates discovery of LANforge devices, allows running simple network tests, and presents results in an easy-to-understand format.

Features

  • Device Discovery: Simulated discovery of LANforge devices on the network.
  • Test Execution: Simulated execution of a network throughput test returning dummy but realistic metrics.
  • Result Visualization: Clear textual display of test results with throughput, latency, and packet loss.
  • User Interface: Clean & responsive UI with interactive controls to trigger discovery and tests.

Tech Stack

  • Backend: Python Flask API serving endpoints for discovery and test execution.
  • Frontend: Plain HTML, CSS, and JavaScript providing user controls and result display.
  • Data: Simulated data used for devices and test results. No database used in this MVP.
